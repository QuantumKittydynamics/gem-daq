Instructions

1. Check that "OptoHybrid > System registers > VFAT2 trigger source" is set to "External".

2. Check that "OptoHybrid > System registers > Clock source" is set to "External".

3. Reset all the counters on the "GLIB" and "OptoHybrid" control pages.

4. Select "Apply default parameters to all VFAT2s" and "Start all VFAT2s" on the "VFAT2s" control page.

5. Mask all the empty VFAT2 slots on the "VFAT2" control page.

6. Perform a "Threshold scan" on every VFAT2 and select a data point in the graph to set the threshold to a given value.

7. Perform a "Latency scan" on every VFAT2 and select a data point in the graph to set the latency to a given value.

8. Run "Start data acquisition" on the "Tracking Data" page to check the collected data.
